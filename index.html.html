Planning Assumptions

Planning logic: gross capacity is reduced by leave/TDY, recurring admin/meetings, default overhead, and planning buffer. The dashboard compares planned workload against adjusted production capacity for the selected planning period. In four-week and monthly views, weekly capacity is multiplied by the number of weeks in the selected period.

Save behavior: entries auto-save to this browser using local storage. Use Export JSON to back up the app data or move it to another computer/browser. CSV export is for reporting, not full restoration.
